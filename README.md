# IPO Information Web Application

A full-stack web application providing detailed, real-time IPO-related information to the public. Built using **React.js** for the frontend, **Express.js** for the backend, and **PostgreSQL** for the database.

This application displays comprehensive IPO data including price bands, issue details, company logos, documents (RHP & DRHP), listing status, and real-time market performance metrics such as CMP and current return. Designed for both public access on the Bluestock website/app and client integrations.

---

## 🧠 Features

- 📈 Live IPO listing dashboard
- 🏢 Company details with logo and listing info
- 💰 Price Band, Issue Size, IPO Price, Listing Price, Listing Gain, CMP
- 📄 RHP & DRHP document download
- 📅 Important IPO dates (Open, Close, Listing)
- 🔁 Real-time CMP and return update
- 📡 REST API for third-party integration (clients' apps/websites)

---

## 📦 Tech Stack

- **Frontend**: [React.js](https://reactjs.org/)
- **Backend**: [Express.js](https://expressjs.com/)
- **Database**: [PostgreSQL](https://www.postgresql.org/)
- **ORM**: [Sequelize](https://sequelize.org/)
